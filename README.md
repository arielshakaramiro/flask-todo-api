# Flask Todo API

API sederhana untuk mengelola Todo list.

## Endpoint

- GET /todos
- GET /todos/<id>
- POST /todos
- DELETE /todos/<id>

## Cara Jalankan

```bash
pip install -r requirements.txt
python app.py
